# Remove all uploaded brand tables

You can remove all tables that were uploaded by brand. To remove all defined tables you need to login as client on StopChurn platform and send following request.

## Request
- Method `DELETE`
- Header `Authorization`
- Route `client-data/client-table/:brandId`

## Response
- 200 OK - Success

- 400 Bad Request - Error during removal

```json
{
  "statusCode": 400,
  "error": "Bad Request",
  "message": "Error while removing all client tables for brandId"
}
```
- 403 Forbidden - Error during database initialization
```json
{
    "statusCode": 403,
    "message": "Invalid token",
    "error": "Forbidden"
}
```
