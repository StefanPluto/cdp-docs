# Database initialization

This endpoint is used to begin database initialization.

## Request
- Method `GET`
- Header `Authorization`
- Route `client-data/initialize-db`

## Response
- 200 OK - Success

- 400 Bad Request - Error during database initialization

```json
{
    "statusCode": 400,
    "message": "Error message",
    "error": "EventSource encountered an error:"
}
```
- 401 Unauthorized - Error during database initialization
```json
{
    "statusCode": 401,
    "message": "Invalid client api token",
    "error": "Unauthorized"
}
```
